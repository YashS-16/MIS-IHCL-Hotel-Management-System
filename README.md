# 🏨 TAJ Hotel Management System

A fully functional **Hotel Management System** built using ReactJS that simulates real-world hotel operations including booking, room management, payments, and billing.

---

## 🚀 Features

### 🔐 Authentication
- Role-based login system:
  - Admin
  - Reception
  - Customer
- Input validation for all login fields

---

### 🧑‍💼 Admin Panel
- Add / Delete Rooms
- Set Room Type & Price
- View All Bookings
- Track Room Availability
- Check-In / Check-Out functionality

---

### 🛎️ Reception Panel
- Create bookings for walk-in customers
- Assign available rooms
- Auto-fetch room type & price
- Redirect to payment

---

### 👤 Customer Panel
- View available rooms in card format
- See room price and availability
- Book rooms online
- View personal booking details

---

### 💳 Payment System
- Multiple payment methods:
  - UPI
  - Card
  - Cash
- Per-booking payment handling
- Payment status tracking (Paid / Pending)

---

### 🧾 Billing System
- Auto-generated invoice
- Displays:
  - Customer details
  - Room details
  - Payment status
  - Total amount

---

### 🔄 Real-Time Logic
- Rooms become unavailable after booking
- Rooms become available after checkout
- Shared data across Admin, Reception & Customer

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- CSS (Modular Styling)
- React Router
- LocalStorage (State Persistence)
